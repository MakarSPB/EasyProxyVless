# EasyProxyVless

Windows‑клиент на .NET 8 (WPF) для проксирования через VLESS (xray-core + wintun).
В репозитории лежат только готовые сборки.

## Требования
- Windows 10/11 x64
- .NET 8 Desktop Runtime (если сборка не self-contained)
- Доступ в интернет для загрузки подписки и обновления xray

## Как использовать
1. Скачайте архив из раздела Releases.
2. Распакуйте в любую папку, запустите `EasyProxyVless.exe` с правами администратора.
3. Вставьте URL подписки, нажмите «Обновить» и запустите выбранный узел.
4. При первом запуске и для обновления xray используйте кнопку «Обновить Xray» в настроках.
